1  Installation and environments

1.1 Install Python

  • Open the python website and install python3 for your OS

  • Choose the most recent version of Python

1.2 Install Pycharm

  • Open Pycharm website. There are different version of Pycharm. I recommend to install the Pycharm professional because it has better syntax highlighting. To have the professional version you need to have an academic email address which is from your University or institute.

  • Alternatively you can install the community version of Pycharm which is free.

  • Run Pycharm

  • Create a new Pycharm project. Choose pure python. Give a name of the project. And Pycharm will create the project for you

1.3 Install oTree

Install oTree in your computer by using the terminal in Pycharm.

  • pip3 install otree

1.3.1 Upgrading/reinstalling oTree

pip3 install -U otree

  • Recommend upgrading every couple of weeks.

1.4 oTree Setup

  • From your command prompt, create your project folder:

otree startproject myproject

  • Move into the folder you just created:

cd myproject

  • Run the server

otree devserver

  • Open your browser to http://localhost:8000/. You should see the oTree demo site.

  • To stop the server, press Control + C at your command line.

  • To create a new app, run otree startapp your_app_name.

  • Session configs are defined in settings.py.
