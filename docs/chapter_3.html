3  More on App in oTree

3.1 App structure

  1. To create an application named game_app move to the oTree folder
  • cd oTree
  1. Create the application
  • otree startapp game_app

  1. Move to the folder game_app

  2. In this folder, you will find the following files as default

  • models.py

  • pages.py

  • tests.py

  1. In this folder, you will also find a sub folder

  • templates/game_app

    • Mypage.html

    • Results.html
